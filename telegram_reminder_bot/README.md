# 🔐 Telegram Reminder Bot with Encrypted Storage

Безопасный Telegram бот с напоминаниями, задачами, заметками и хранилищем паролей.

**Все данные зашифрованы AES-256-GCM и хранятся в JSON формате.**

## ✨ Возможности

### 🔔 Напоминания
- Создание напоминаний на естественном языке
- **Постоянные уведомления** со звуком пока не отреагируете
- Откладывание (snooze) на 5-120 минут или до завтра
- Повторяющиеся: ежедневно, еженедельно, ежемесячно, ежегодно

### ✅ Список задач (TODO)
- Приоритеты: 🟢 низкий, 🟡 средний, 🟠 высокий, 🔴 срочный
- Статусы: ожидает, в работе, выполнено
- Дедлайны с уведомлениями

### 📝 Зашифрованные заметки
- Создание и редактирование заметок
- Закрепление важных заметок
- **Полное шифрование содержимого AES-256-GCM**

### 🔐 Хранилище паролей
- Безопасное хранение логинов и паролей
- **Шифрование AES-256-GCM** (военный стандарт)
- Генератор надёжных паролей (12-32 символа)
- Поиск по сервисам
- Избранное для быстрого доступа

### 🔄 P2P Синхронизация
- Синхронизация между устройствами
- Данные остаются зашифрованными при передаче
- Сервер никогда не видит открытые данные

## 🔒 Безопасность

### Шифрование AES-256-GCM
- **Алгоритм:** AES-256-GCM (Galois/Counter Mode)
- **Длина ключа:** 256 бит
- **Аутентификация:** Встроенная (AEAD)
- **Уникальный nonce** для каждой операции шифрования

### Ключ шифрования
- Генерируется из вашего Telegram ID
- Или из мастер-пароля с PBKDF2 (600,000 итераций)
- Ключ никогда не покидает ваше устройство

### Хранение данных
```
data/
  └── storage.sqlite3               # Зашифрованное SQLite-хранилище
```

Структура зашифрованного файла:
```json
{
  "version": "2.0",
  "algorithm": "AES-256-GCM",
  "key_fingerprint": "a1b2c3d4...",
  "data": "BASE64_ENCRYPTED_DATA..."
}
```

## 🚀 Установка

### 1. Клонирование
```bash
cd telegram_reminder_bot
```

### 2. Виртуальное окружение
```bash
python -m venv venv
source venv/bin/activate  # Linux/macOS
# или
venv\Scripts\activate  # Windows
```

### 3. Зависимости
```bash
pip install -r requirements.txt
```

### 4. Конфигурация
```bash
cp .env.example .env
```

Отредактируйте `.env`:
```env
BOT_TOKEN=ваш_токен_от_BotFather
DATA_DIR=./data
TIMEZONE=Europe/Moscow

# P2P синхронизация (опционально)
P2P_ENABLED=false
P2P_PORT=8765
P2P_SECRET=ваш_секретный_ключ
```

### 5. Запуск
```bash
python bot.py
```

## 🔊 Постоянные уведомления

Когда наступает время напоминания:
1. 🔔 Бот отправляет уведомление со звуком
2. 🔄 Если не ответили — повторяет каждые 60 секунд
3. Варианты действий:
   - ✅ **Выполнено** — завершить
   - ⏸️ **Отложить** — на 5/10/15/30 мин, 1-2 часа, до завтра
   - 🔇 **Без звука** — продолжить без звука

## 🔄 P2P Синхронизация

Для синхронизации между устройствами:

1. Включите P2P в `.env`:
```env
P2P_ENABLED=true
P2P_PORT=8765
P2P_SECRET=ваш_секретный_ключ
```

2. Бот запустит сервер синхронизации

3. Данные синхронизируются в зашифрованном виде

**Важно:** Сервер никогда не расшифровывает данные!

## 🛡️ Технические детали шифрования

### AES-256-GCM
```python
# Алгоритм шифрования
from cryptography.hazmat.primitives.ciphers.aead import AESGCM

key = os.urandom(32)  # 256 бит
nonce = os.urandom(12)  # 96 бит (уникальный для каждой операции)
aesgcm = AESGCM(key)
ciphertext = aesgcm.encrypt(nonce, plaintext, None)
```

### PBKDF2 для паролей
```python
# Деривация ключа из пароля
from cryptography.hazmat.primitives.kdf.pbkdf2 import PBKDF2HMAC

kdf = PBKDF2HMAC(
    algorithm=SHA256(),
    length=32,
    salt=salt,
    iterations=600000  # Высокое число итераций
)
key = kdf.derive(password)
```
