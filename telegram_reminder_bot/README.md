# 🔔 Telegram Reminder Bot

Бот для Telegram с напоминаниями, списком задач (TODO) и повторяющимися напоминаниями.

## ✨ Возможности

### 🔔 Напоминания
- Создание напоминаний с естественным языком ввода
- **Постоянные уведомления** - бот будет напоминать каждую минуту, пока вы не отреагируете
- Уведомления со звуком
- Возможность отложить (snooze) на выбранное время
- Отметка о выполнении

### 🔄 Повторяющиеся задачи
- Ежедневные напоминания
- Еженедельные напоминания
- Ежемесячные напоминания
- Ежегодные напоминания
- Пользовательский интервал повторения

### 📋 TODO список
- Создание задач
- Приоритеты (низкий, средний, высокий, срочный)
- Статусы (ожидает, в работе, выполнено)
- Дедлайны с уведомлениями
- Статистика выполнения

## 🚀 Установка

### 1. Клонирование репозитория

```bash
cd telegram_reminder_bot
```

### 2. Создание виртуального окружения

```bash
python -m venv venv
source venv/bin/activate  # Linux/macOS
# или
venv\Scripts\activate  # Windows
```

### 3. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 4. Настройка переменных окружения

Скопируйте файл `.env.example` в `.env` и заполните:

```bash
cp .env.example .env
```

Отредактируйте `.env`:

```env
BOT_TOKEN=ваш_токен_бота_от_BotFather
DATABASE_URL=sqlite+aiosqlite:///./reminder_bot.db
TIMEZONE=Europe/Moscow
```

### 5. Запуск бота

```bash
python bot.py
```

## 📝 Примеры использования

### Создание напоминания

Бот понимает естественный язык:

```
Позвонить маме завтра в 10:00
Встреча через 2 часа
Оплатить счёт 15 января в 12:00
Забрать посылку в пятницу
Позвонить врачу через 30 минут
```

### Поддерживаемые форматы времени

- `через 5 минут` / `через 2 часа` / `через 3 дня`
- `завтра` / `послезавтра`
- `завтра в 10:00` / `сегодня в 18:30`
- `в понедельник` / `в пятницу в 9:00`
- `15 января в 12:00`
- `10.01.2025 14:30`
- `14:30` (на сегодня или завтра)

## 🔊 Постоянные уведомления

Когда наступает время напоминания:

1. Бот отправляет уведомление со звуком
2. Если вы не отреагировали - повторяет каждые 60 секунд
3. Вы можете:
   - ✅ **Выполнить** - отметить напоминание выполненным
   - ⏸️ **Отложить** - на 5/10/15/30 мин, 1/2 часа или до завтра
   - 🔇 **Отключить звук** - уведомления продолжатся, но без звука

## 🔄 Повторяющиеся напоминания

После создания напоминания можно настроить повторение:

- **Без повтора** - одноразовое напоминание
- **Ежедневно** - каждый день в то же время
- **Еженедельно** - каждую неделю в тот же день
- **Ежемесячно** - каждый месяц в ту же дату
- **Ежегодно** - каждый год в ту же дату

Когда вы отмечаете повторяющееся напоминание выполненным, автоматически создаётся следующее.

## ⚙️ Конфигурация

В файле `config.py` можно настроить:

- `PERSISTENT_REMINDER_INTERVAL` - интервал между постоянными уведомлениями (по умолчанию 60 секунд)
- `MAX_SNOOZE_TIMES` - максимальное количество откладываний
- `DEFAULT_SNOOZE_MINUTES` - время откладывания по умолчанию
- `DEFAULT_TIMEZONE` - часовой пояс по умолчанию

## 🛡️ Безопасность

- Все данные хранятся в локальной SQLite базе данных
- Каждый пользователь видит только свои напоминания и задачи
- Токен бота хранится в переменных окружения
